# Keyboard Shortcuts

### Command Palette

When you are in the console, use the command palette to easily look up keyboard shortcuts for various actions.

| Action                 | Mac                | Windows/Linux            |
| ---------------------- | ------------------ | ------------------------ |
| Access Command Palette | ⌘ + K **OR** ⌘ + P | ctrl + K **OR** ctrl + P |

### Cell Controls

| Action                                                                                           | Mac                                                                        | Windows/Linux                                                              |
| ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
