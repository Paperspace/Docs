# Show Job info

## About

Show job information for the job with the given id.

## Example Use

```text
$ gradient jobs show --jobId "j123abc"
```

### **Properties**

| Name |  | Type | Description |
| :--- | :--- | :--- | :--- |
| `jobId` |  | string | Id of the job to show |
